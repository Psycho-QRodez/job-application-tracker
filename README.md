# Job Application Tracker

A full-stack job application tracking system built with Next.js and Supabase.

## Features

- Add job applications
- Update application status
- Delete applications
- Search by company
- Filter by application status
- Persistent PostgreSQL database storage

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
